# Smart-Recipe-Generator

live at https://alokranjan42.github.io/Smart-Recipe-Generator/

### Project Overview ###
Smart Recipe Generator is a full-stack inspired AI web application that helps users create recipes based on the ingredients they already have. The main goal of the project is to solve a common everyday problem—people often struggle to decide what to cook with limited ingredients. This application allows users to enter available ingredients and instantly generates creative, easy-to-follow recipes using Google’s Gemini AI.


## Key Features ##
It is based on client external Api Architecure.
It has JWT based Secure Authentication System that secures your saved recipe .it makes accessiblity of the recipes.
It uses Google Gemini API service to detect the user’s ingredients and generate personalized recipes in real time.



### Tech Stack ##
